# Selenium-Spider_Code
These codes are used for crawl data fromTianyancha.com，the login part mainly depends on using the local cookies，so you need to have your own account.


Before you use selenium, make sure you have installed the chrome web-driver.
First, you need to use 获取天眼查cookie.py, this program will automatically open the broswer and wait 20 seconds for you to enter account and password, then the cookies will be saved as a txt file in the
local dir.
